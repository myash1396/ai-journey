PROJECT: LOAN STATUS CHECKER
VERSION: 1.0

1. OVERVIEW
Bank needs a simple system extension in their existing application for customers to check their loan application status online.

2. REQUIREMENTS
2.1 Customer enters their loan application ID on the web portal - new UI.
2.2 System displays current status: Submitted, Under Review, Approved, Rejected.
2.3 If approved, show loan amount and disbursement date.
2.4 If rejected, show reason for rejection.
2.5 Customer receives SMS notification when status changes.

3. USERS
- Retail customers checking their own loan status
- Bank staff updating loan status

4. NON FUNCTIONAL
- System must respond within 3 seconds
- Available 24/7
